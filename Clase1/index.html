Introducción

¿Qué es AngularJS?

AngularJS es un Framework para crear páginas web dinámicas.

Es un framework
No es simplemente una biblioteca de funciones sino un framework, ya que proporciona una estructura o arquitectura para el desarrollo de aplicaciones
Open Source
Escrito en Javascript lo que permite la extensibilidad.
Creado y mantenido por Google
Esta característica es una garantía de continuidad.
Patrón MVC (MV*)
Modularidad, separación de responsabilidades, mejor extensibilidad, y otras ventajas asociadas a este patrón.
Permite extender la sintaxis HTML
Permite crear etiquetas y atributos HTML con funcionalidad específica.
Apropiado para SPA
Nos proporciona las herramientas necesarias para construir aplicaciones SPA, como routing, plantillas y vistas parciales, etc.
Muy popular
Gran comunidad de desarrolladores, recursos y ejemplos en Internet.

  • Inyección de dependencias

  • Controladores
  • Scope
  • Enrutamiento
  • Plantillas

Prerrequisitos
