Clase 4 - Introducción al enlace de datos

El enlace de datos o data binding consiste en emparejar la vista con el modelo de manera que los cambios en uno de ellos se reflejen inmediatamente en el otro.

En el ejemplo a continuación se muestra cómo enlazar un input con una variable javascript del contexto, y cómo mostrarla usando la sintáxis de doble llave. 

<input type="text" ng-model="nombre"/>
<span>{{nombre}}</span>
{{nombre}}
