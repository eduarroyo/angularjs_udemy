Clase 3 - Directivas

Las directivas son la herramienta que utilizamos para extender el HTML. Algunos ejemplos de directivas son:

  • ng-app para inicializar la aplicación. Se coloca en la etiqueta que contiene la aplicación, puede ser html, body o div. 
    <!--Sin identificador del módulo:-->
<html ng-app>
	<head>...</head>
	<body>...</body>
</html>
    <!--Con identificador del módulo:-->
<html ng-app="miAplicacion">
	<head>...</head>
	<body>...</body>
</html>
  • ng-model para declarar una variable en el modelo. Se emplea en las etiquetas input, select y textarea. El valor del elemento del dom y de la variable siempre está sincronizado (bind). 
    <input ng-model="miVariable"/>
